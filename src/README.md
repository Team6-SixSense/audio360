# Audio360 Source Code

This directory contains the source code for Audio360.

## Folder Structure

The folders and files for this project are as follows:

    .
    ├── audio360_viz        # Android application to deploy to glasses.
    ├── cmake               # Cmake specific to hardware build.
    ├── configs             # Configuration files.
    ├── features            # Audio360 software features.
    ├── hardware_interface  # Hardware interfaces code. Contains Hardware Abstraction Layer.
    ├── helper              # Helper code.
    ├── lib                 # Libraries. Each library is a submodule.
    ├── prototyping         # Prototyping software. Code in this directory is not compiled.
    └── runtimes            # Runtime software. Allows to build different modes.
