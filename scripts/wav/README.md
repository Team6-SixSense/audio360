# WAV Converter

To see usage, run

```bash
python wav_converter.py -h
```

## Input file format

The data in the input file must be represented as PCM (Pulse Code Modulation).
That is data is in range of [-1.0, 1.0].

Each row represents data from a single timestep.

```text
0.000000000000000000e+00
1.250505236945280985e-01
2.481378479437378881e-01
3.673295940613788280e-01
4.807545410165317001e-01
5.866320022005455659e-01
6.832997808714387222e-01
7.692402653962487902e-01
8.431042546155974593e-01
9.037321392901134232e-01
9.501721070958867310e-01
```
