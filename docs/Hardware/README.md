# Hardware Documentation

This folder contains hardware documentation provided by the manufacturer, STMicroelectronics (STM), as well as team specific hardware documentation.

This project is uses STM's [NUCLEO-144](https://www.st.com/en/evaluation-tools/nucleo-f767zi.html) microcontroller with the [STM32F767ZI MCU](https://www.st.com/en/microcontrollers-microprocessors/stm32f767zi.html) 

| File | Description | Author |
|------|-------------| ------ |
| NUCLEO-144-PINOUTS.pdf | Snapshot of STM's manual of the microcontroller's pinouts | STM |
| NUCLEO-144-BOARD.pdf | STM's Nucleo microcontroller user manual | STM |
| STM32-PINOUT.png | Image of the microcontroller's pinouts | STM |
| stm32f767zi.pdf | The microcontroller's MCU's user manual | STM |
