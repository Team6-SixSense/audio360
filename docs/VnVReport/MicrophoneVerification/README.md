# Microphone Verification

This folder contains microphone verification results.
The verification checks to see whether the microphones can capture audio data from the environment.

## Verification Process

1. Collect data from microphones and store as raw bytes on a SD card.
2. Convert the raw audio data to a .wav file.
3. Play the .wav file and listen to ensure the recorded audio is what is expected.
4. Analyze the .wav file in an audio editor tool such as Audacity.
