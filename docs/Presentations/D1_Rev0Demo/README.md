# REV0 Presentation

Rev 0 demonstrated the product built on a breadboard with all core features such as DoA and classification running.
The software runs on the microcontroller and communicates to the glasses which display the results on the frames.
`rev0_demo.mp4` is a video recording the team has captured on the glasses with Audio360 running.

During the presentation, the team allowed the professor and TA to try the glasses while Audio360 was running.

See the [Rev 0 Presentation meeting notes](https://github.com/Team6-SixSense/audio360/issues/484) on feedback given by the professor and TA.
