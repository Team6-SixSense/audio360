# Extras

Extras are supplementary documents that provide additional insight into the project. The following reports are included:

1. Hardware Report: Describes the hardware components used in the project, their underlying principles, and their impact on software design decisions.
   - See `Hardware` folder for more information.
2. Software Theory Report: Details the development of the audio processing algorithms, including the Direction of Arrival and audio classification methods, as well as the team’s process of experimentation and iterative refinement.
    - See `Software` folder for more information.
