# Code of Conduct

## Purpose

This project aims to develop an embedded system for smart glasses that improves situational awareness for people who are hard of hearing.
A respectful, inclusive, and collaborative environment is essential to achieving this goal.
This Code of Conduct establishes standards of behavior for all contributors and participants.

## Expected Behavior

All contributors are expected to:

- Treat others with respect, professionalism, and courtesy.
- Communicate constructively and thoughtfully, especially during technical discussions and code reviews.
- Be inclusive and considerate of diverse perspectives, backgrounds, and experiences.
- Provide helpful feedback focused on improving the project and its quality.
- Follow ethical engineering practices, including honesty in reporting results and limitations.

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or offensive comments based on disability, race, gender, age, religion, or any other personal characteristic.
- Personal attacks, insults, or demeaning language.
- Intimidation, threats, or disruptive behavior.
- Any conduct that undermines collaboration or creates a hostile environment.
- Using the software to collect data of users.

Given the projectâ€™s focus on assistive technology for people who are hard of hearing, special consideration should be given to:

- Clear and accessible documentation.
- Inclusive language in code, comments, and discussions.
- Design decisions that respect user dignity, privacy, and autonomy.
- Testing and feedback processes that consider real-world user needs.

## Responsibilities

Project maintainers are responsible for:

- Enforcing this Code of Conduct fairly and consistently.
- Addressing reported issues promptly and respectfully.
- Taking appropriate corrective action when violations occur.

## Enforcement

If unacceptable behavior occurs, maintainers may take one or more of the following actions:

- Issue a warning.
- Request an apology or corrective action.
- Temporarily restrict participation.
- Remove a contributor from the project.

The severity of the response will depend on the nature and frequency of the violation.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repository, issues, pull requests, and discussions.
- Project documentation.
- Any official project meetings or demonstrations related to this capstone project.

## Acknowledgment

By contributing to this project, participants agree to abide by this Code of Conduct and support a respectful and inclusive development environment.
